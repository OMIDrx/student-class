# Student Grade Calculator

A simple Python project that demonstrates object-oriented programming (OOP) by creating a `Student` class capable of calculating the average score and evaluating academic performance.

## Features

- Object-Oriented Programming (OOP)
- Calculate average score
- Evaluate student performance
- Simple and beginner-friendly code

## Technologies

- Python

## Project Structure

```
student-grade-calculator/
├── student.py
└── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/student-grade-calculator.git
```

Move into the project directory:

```bash
cd student-grade-calculator
```

Run the program:

```bash
python student.py
```

## Example

```python
student = Student("Omid", [18, 20, 17, 19])

print(student.average())
print(student.state())
```

Output:

```
18.5
Very Good
```
